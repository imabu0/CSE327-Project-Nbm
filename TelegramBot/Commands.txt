Commands for File Search Bot

1. /start
   - Description: Starts the bot and provides a welcome message with instructions.
   - Usage: /start
   - Response: 
     "Hello! Welcome to the File Search Bot.

      Here's how to use me:
      /start - Start the bot
      /help - Show help message
      Send a keyword to search for files or images.
      /search <file_type> <keyword> - Search for specific file types (e.g., /search txt hello)
      /date <YYYY-MM-DD> - Search for files modified on a specific date
      /size <>=< <size_in_bytes> - Search for files by size (e.g., /size > 1000000)
      /image <keyword> - Search for images by keyword."

2. /help
   - Description: Provides a detailed guide on how to use the bot.
   - Usage: /help
   - Response:
     "Here's how to use the bot:
      /start - Start the bot
      /help - Show this help message
      Send a keyword to search for files or images.
      /search <file_type> <keyword> - Search for specific file types (e.g., /search txt hello)
      /date <YYYY-MM-DD> - Search for files modified on a specific date
      /size <>=< <size_in_bytes> - Search for files by size (e.g., /size > 1000000)
      /image <keyword> - Search for images by keyword."

3. Send a Keyword
   - Description: Searches for files or images containing the keyword in their names or content.
   - Usage: Send any keyword (e.g., "report").
   - Response:
     - If files or images are found, the bot will send them.
     - If no files or images are found, the bot will reply:
       "No files found for 'your_keyword'."

4. /search <file_type> <keyword>
   - Description: Searches for specific file types containing the keyword.
   - Usage: /search <file_type> <keyword> (e.g., /search txt hello)
   - Response:
     - If files are found, the bot will send them.
     - If no files are found, the bot will reply:
       "No <file_type> files found for '<keyword>'."

5. /date <YYYY-MM-DD>
   - Description: Searches for files modified on a specific date.
   - Usage: /date <YYYY-MM-DD> (e.g., /date 2023-10-01)
   - Response:
     - If files are found, the bot will send them.
     - If no files are found, the bot will reply:
       "No files found for date 'YYYY-MM-DD'."

6. /size <>=< <size_in_bytes>
   - Description: Searches for files larger, smaller, or equal to a specified size.
   - Usage: /size <>=< <size_in_bytes> (e.g., /size > 1000000)
   - Response:
     - If files are found, the bot will send them.
     - If no files are found, the bot will reply:
       "No files found for size <operator> <size_in_bytes> bytes."

7. /image <keyword>
   - Description: Searches for images containing the keyword in their names.
   - Usage: /image <keyword> (e.g., /image cat)
   - Response:
     - If images are found, the bot will send them.
     - If no images are found, the bot will reply:
       "No images found for '<keyword>'."

---

Notes:
- Replace <file_type>, <keyword>, <YYYY-MM-DD>, <>=<, and <size_in_bytes> with actual values.
- The bot only responds to authorized users. Unauthorized users will receive:
  "You are not authorized to use this bot."
